A bit about the team

In the penguin hall Web development class we are dedicated to honoring and memorializing Bob Ross, the American painter, art instructor, and television host.
Our amazing coding team, consisting of Nathalie Forsythe, Mattea Kaiser, Masha Gilberg, Katie Riley, Crissy Rietti, Alea Jalal, and Annie Ainsworth grouped together for their final project of the semester to create the tribute site. This website was created using Visual Studio Code editor and Github as our platform to collaborate on the project.
